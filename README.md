# 📊 Sales Dashboard (Power BI)

## 📌 Overview
This project presents an interactive sales dashboard built using Power BI. The goal is to visualize key business metrics and uncover insights related to sales performance, profitability, product trends, and regional performance.

---

## 🛠 Tools Used
- Microsoft Power BI
- CSV Dataset (Superstore Sales)

---

## 📂 Dataset
The dataset contains retail sales data including:
- Order Date
- Product Name
- Category and Sub-Category
- Sales and Profit
- Region

---

## 📊 Dashboard Features

### 🔹 Key Performance Indicators (KPIs)
- Total Sales
- Total Profit

### 🔹 Visualizations
- Sales by Category
- Profit by Region
- Sales Trend Over Time (2014–2017)
- Top 5 Products by Sales

## 🧠 Key Insights
- A small number of products contribute significantly to total revenue.
- The West region generates the highest profit among all regions.
- Sales show an increasing trend over time, especially after 2015.
- Some categories outperform others consistently in revenue generation.
